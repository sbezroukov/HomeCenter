Q: Дайте определение уравнения с двумя переменными. Приведите три примера уравнений разной степени.

Q: Что называется решением уравнения с двумя переменными? Покажите на примере уравнения 3x - 2y = 5, что пара (3; 2) является решением, а пара (1; 1) — нет.

Q: Как определить степень уравнения с двумя переменными? Найдите степень уравнений: а) x² + 4xy = 5; б) (x + y)² = x⁶ - 1.

Q: Что такое график уравнения с двумя переменными? Почему график линейного уравнения всегда является прямой линией?

Q: Постройте график уравнения 2x - y = 4. Какие точки принадлежат графику: (0; -4), (2; 0), (1; 2)?

Q: Объясните, почему графиком уравнения x² = y² является пара прямых y = x и y = -x, а не одна линия.

Q: Какие особенности имеют графики уравнений: а) 3x + 0y = 12; б) 0x + y = 1; в) (x - 2)(y - 3) = 0?

Q: Приведите примеры известных кривых (лемниската Бернулли, лист Декарта) и их уравнений. Почему графики уравнений с двумя переменными могут быть такими разнообразными?

---

ОТВЕТЫ:
1. Уравнение с двумя переменными — равенство, содержащее две переменные (обычно x и y), которое обращается в верное равенство при определённых значениях этих переменных. Примеры: 2x + 3y = 6 (1-я степень), x² + y² = 25 (2-я степень), xy = 12 (2-я степень).
2. Решение — упорядоченная пара чисел (x; y), обращающая уравнение в верное равенство. Для 3x - 2y = 5: при (3; 2) → 3·3 - 2·2 = 9 - 4 = 5 — верно; при (1; 1) → 3·1 - 2·1 = 1 ≠ 5 — неверно.
3. Степень — наибольшая сумма показателей степеней переменных в одночлене после приведения к стандартному виду (правая часть = 0). а) x² + 4xy - 5 = 0 → степень 2; б) (x+y)² - x⁶ + 1 = 0 → x² + 2xy + y² - x⁶ + 1 = 0 → степень 6.
4. График — множество точек координатной плоскости, координаты которых являются решениями уравнения. Для линейного уравнения ax + by = c (a, b ≠ 0) график — прямая, так как уравнение можно привести к виду y = kx + b (линейная функция).
5. Выразим y: y = 2x - 4. Точки: (0; -4) → -4 = -4 — да; (2; 0) → 0 = 0 — да; (1; 2) → 2 = -2 — нет.
6. x² = y² → x² - y² = 0 → (x - y)(x + y) = 0 → x - y = 0 или x + y = 0 → y = x или y = -x. Это две прямые, пересекающиеся в начале координат.
7. а) 3x = 12 → x = 4 — вертикальная прямая; б) y = 1 — горизонтальная прямая; в) (x-2)(y-3)=0 → x=2 или y=3 — две перпендикулярные прямые, образующие «крест».
8. Лемниската Бернулли: (x² + y²)² = 2a²(x² - y²) — кривая в форме восьмёрки. Лист Декарта: x³ + y³ = 3axy — кривая с петлёй. Разнообразие графиков связано с возможностью комбинировать переменные в разных степенях и операциях.
