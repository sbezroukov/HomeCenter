Q: Как связаны уравнения вида x² + px + q = 0 и разложение на множители (x - x₁)(x - x₂) = 0? Докажите это на примере уравнения с корнями (√3-1)/2 и (√3+1)/2.

---

ОТВЕТЫ:
1. ОДЗ: x ≠ 5, x ≠ -5. Общий знаменатель (x-5)(x+5). Умножаем: (x-4)(x+5) + (x-6)(x-5) = 2(x²-25). Раскрываем: x²+x-20 + x²-11x+30 = 2x²-50 → 2x²-10x+10 = 2x²-50 → -10x = -60 → x=6. Проверка: 6≠±5 → корень. Ответ: x=6.
2. ОДЗ: x ≠ -1, x ≠ 2, x ≠ 0. Общий знаменатель x(x+1)(x-2). После умножения и упрощения получаем квадратное уравнение 5x²-23x+12=0 → x₁=4, x₂=0,6. Оба входят в ОДЗ. Ответ: 0,6; 4. ОДЗ нужен, чтобы сразу исключить недопустимые значения и не получить посторонние корни.
3. Раскроем правую часть: 10(x-0,1)(x+2) = 10(x²+2x-0,1x-0,2) = 10(x²+1,9x-0,2) = 10x²+19x-2 — совпадает с левой частью. Это тождество показывает, что если квадратный трёхчлен имеет корни x₁, x₂, то ax²+bx+c = a(x-x₁)(x-x₂).
4. Если дискриминант D = b²-4ac является полным квадратом целого числа, и коэффициенты a, b, c целые, то корни рациональны, и трёхчлен раскладывается на множители с целыми коэффициентами.
5. График y=6/x — гипербола, y=-x+6 — прямая. Пересекаются в двух точках → два корня. Без графика: после приведения к общему знаменателю получаем квадратное уравнение x²-6x+6=0, D=36-24=12>0 → два корня.
6. а) Общий знаменатель (x-1)(x+1)=x²-1: (x+1+x-1)/(x²-1)=10/(x²-1) → 2x=10 → x=5 (проверка: 5≠±1). б) Замена не упрощает. Первый метод эффективнее.
7. Многоэтажные дроби легко приводят к ошибкам при раскрытии скобок. Нужно действовать по шагам: сначала упростить внутренние дроби, затем внешние. Проверка каждого преобразования обязательна.
8. По теореме Виета: для x²+px+q=0 сумма корней = -p, произведение = q. Если корни x₁, x₂, то (x-x₁)(x-x₂)=x²-(x₁+x₂)x+x₁x₂ = x²+px+q. Для корней (√3±1)/2: сумма = √3, произведение = (3-1)/4=0,5. Уравнение: x²-√3x+0,5=0 → 2x²-2√3x+1=0.
